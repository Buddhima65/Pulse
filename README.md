# Pulse - Android Wellness & Productivity App

## 📱 Overview

Pulse is a modern Android wellness and productivity application developed using **Kotlin** and **Android Studio**. The app is designed to help users build healthy daily routines, track emotional well-being, and maintain hydration habits through a clean and interactive mobile experience.

The project focuses on modern Android development practices, responsive UI design, and user-friendly navigation while implementing practical mobile application features.

---

## ✨ Features

### 🧠 Habit Tracking

* Create and manage daily habits
* Track habit completion progress
* Interactive and dynamic habit lists
* Simple and clean productivity-focused interface

### 😊 Mood Journaling

* Record daily moods and personal notes
* Maintain emotional wellness logs
* View mood history and trends
* Graphical mood analytics visualization

### 💧 Water Intake Reminder

* Smart hydration reminder notifications
* Custom reminder intervals
* Daily water intake tracking
* Notification-based user engagement

### ⚙️ Settings & Personalization

* Notification controls
* User preference management
* Lightweight local data persistence using SharedPreferences

### 🎨 Modern UI/UX

* Material Design inspired interface
* Responsive layouts
* Smooth navigation using Bottom Navigation & Fragments
* Clean and minimal mobile experience
